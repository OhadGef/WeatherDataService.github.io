C# Project Weather Data Service

introduction:


We are students in Shenkar College Of Engineering.
This is a final project in C# course that we took in the Third year.
In this project we have developed a library that provides Weather Data Service for other developers,
The developer need to choose the location of the country and city and the code gets the weather data from 2 APIs and show it to the Concole.


github URL:https://github.com/OhadGef/WeatherDataService.github.io

YouTube URL: https://github.com/OhadGef/WeatherDataService.github.io



API exemple:


// set a new Location for the Weather
Location location = new Location();
location.cityName = "Tel Aviv";
location.countryName = "IL";

// chosse the web site of tht API (WORLDWEATHERONLINE or OPENMAP)
IWeatherDataService service =
WeatherDataServiceFactory
.GetWeatherDataService(WeatherDataServiceFactory.ServiceType.WORLDWEATHERONLINE OR OPENMAP);

WeatherData weatherData = service.GetWeatherData(location);
Console.WriteLine(weatherData.ToString());
